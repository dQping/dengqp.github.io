这种方式有两个问题：

  • 因为箭头函数总是匿名的，如果你打算移除监听事件，是做不到的。
  • 如果这个回调函数作为一个属性值传入低阶组件，这些低阶组件在当前组件重新渲染时，可能会进行额外的重新渲染。我们通常建议在构造函数中绑定或使用属性初始化器语法来避免这类性能问题。
