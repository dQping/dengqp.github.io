类型转换

数据类型分为原始数据类型和对象数据类型，本篇主要记录显示类型转换的一些相关知识。JavaScript 语言的每一个值，都属于某一种数据类型。JavaScript 的数据类型，共有六种（ES6 又新增了第七种 Symbol 类型的值）。

  • 原始数据类型：Boolean Null Undefined Number String Symbol
  • 对象数据类型：Object

显示类型转换：

  • Number 函数
  • String 函数
  • Boolean 函数

Number 函数

1、原始类型转换：

  • 数值： 转换后还是数值
  • 字符串： 如果可以被解析为数值，则转换为相应的数值，否则得到 NaN, 空字符串转为 0
  • boolean： true 转换为 1， false 转成 0
  • undefined：转成 NaN
  • null：转成 0

2、对象类型转换：

先调用对象自身的 valueOf 方法，如果返回原始类型的值，再使用 Number();如果返回复合类型，再调用自身的 toString()

如果 toString() 返回原始类型， 再使用 Number() ，如果返回的还是复合类型，则报错。

String 函数

1、原始类型转换：

  • 数值： 转换后是相应的字符串
  • 字符串： 转换后还是原来的值
  • boolean： true 转换为 “true”， false 转成 “false”
  • undefined：转成 “undefined”
  • null：转成 “null”

2、对象类型转换：

先调用对象自身的 toString() 方法，如果返回原始类型的值，再使用 String();如果返回复合类型，再调用自身的 valueOf()

如果 valueOf() 返回原始类型， 再使用 String() ，如果返回的还是复合类型，则报错。

Boolean 函数:

1、原始类型转换：

undefined null +0 -0 空字符串 NaN 返回 false，其他值为 true

隐式类型转换：

  • 四则运算
  • 判断语句
  • Native 调用（console.log(),alert()）

常见题目：

1、[]+[] // “”
2、[]+{} // “[object Object]”
3、{}+[] // 0
4、{}+{} // “[object Object][object object]”
5、true + true // 2
6、1+{a:1} // “1[object Object]”

typeof

  • Undefine -> “undefined”
  • Null -> “object”
  • Boolean -> “boolean”
  • Number -> “number”
  • String -> “string”
  • Symbol -> “symbol”
  • Hot object -> “Implementation-dependent”
  • Function object -> “function”
  • Any other object -> “object”
