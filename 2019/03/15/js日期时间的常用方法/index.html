js日期时间的常用方法

因为这两天在改写日期选择插件，所以记录一下针对日期的常用方法，以及可能用到的操作处理,方便以后需要的时候可以查阅。

常用的方法

方法：

创建日期对象

// 不传参数表示获取当时间
var now = new Date();

// 传字符串日期格式 'YYYY/MM/DD' 或 'YYYY-MM-DD' 或 'YYYY.MM.DD'
// 
var myDate = new Date('2018/1/28');

// 传秒数，一般用于将时间戳转化为日期
var myDate = new Date(1517414400000);

注： 通过 new Date(字符串日期) 创建date 实例时，字符串日期 ‘YYYY-MM-DD’ 和 ‘YYYY.MM.DD’ 会存在兼容问题，在 safari 中无效，建议都用正则转为 ‘YYYY/MM/DD’ 格式：dateString.replace(/-|./g, ‘/‘)
