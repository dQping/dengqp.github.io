babel 处理 es6+ 语法

  • babel-loader 实际上是 webpack 和 babel 做通信的一个桥梁，但 babel 并不会帮你把 js 中的 es6+ 语法翻译成 es5 语法。
  • @babel/core 是 babel 的一个核心库，它能够让 babel 去识别 js 中的内容，把 js 代码转换成 AST 抽象语法树，然后再把 AST 抽象语法树编译转换出新的语法
  • @babel/preset-env 包含了所有将 es6+ 翻译成 es5 的规则
