基本概念

盒子模型有两种，分别是 ie 盒子模型和标准 w3c 盒子模型

标准模型盒IE模型的区别

主要区别再于宽高计算不同：

  • 标准盒模型：content
  • IE模型： content + padding + border

css如何设置这两种模型

  • 标准盒模型：box-sizing: content-box
  • IE盒模型：box-sizing: border-box

js 如何设置获取盒模型对应的宽高

根据盒模型解释边距重叠

  • 父子元素边距重叠
  • 兄弟元素边距垂直重叠：取最大值
  • 空元素边距重叠：取最大值作为边距

BFC(边距重叠解决方案)

BFC基本概念：块级格式化上下文

BFC渲染规则：

  • 垂直方向边距不会发生重叠；
  • BFC区域不会与浮动元素相重叠
  • BFC作为独立容器，外部元素和里面元素互不影响
  • 计算BFC高度的时候，浮动元素也会参与计算

如何创建BFC：

  • float不为none
  • position 不为 static 或 relative
  • display: table/table-cell/table相关
  • overflow 不为 visible
